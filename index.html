Calculadora React - Eduardo Matos Costa

Sobre o projeto

Esta calculadora foi desenvolvida em React, utilizando componentes funcionais, hooks de estado (useState) e manipulação de eventos. O layout e as funcionalidades foram inspirados na calculadora do Windows, incluindo operações básicas, porcentagem, inversão de sinal, CE, C e backspace.

O projeto utiliza Vite para desenvolvimento rápido e moderno em React. O CSS foi customizado para um visual escuro, moderno e responsivo.

Sinta-se à vontade para explorar e experimentar a calculadora React!
